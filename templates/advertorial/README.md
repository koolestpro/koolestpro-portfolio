# Advertorial Templates

Advertorial templates are long-form editorial sales pages. They are copy-led and should be used when the product needs education, objection handling, and trust-building before the CTA.

## How To Use `ad-01-editorial-story`

Use this template when the page should feel like a sponsored article or editorial recommendation. The copy should follow this argument:

1. Name the problem the buyer already feels.
2. Explain why common fixes fail.
3. Introduce a new mechanism or product insight.
4. Show why this product is different.
5. Prove the claim with real evidence.
6. Repeat the offer at natural decision points.
7. Close with the strongest guarantee, discount, or availability CTA.

Required product inputs are documented in `ad-01-editorial-story/BUILD-NOTES.md`.

## Image Requirements

Every product using this advertorial should have:

| Image | Use |
|---|---|
| `hero.jpg` | Article hero image |
| `problem.jpg` | Problem or old-way image |
| `product.png` | Product cutout for sticky CTA |
| `lifestyle.jpg` | In-use editorial image |
| `detail.jpg` | Mechanism, material, ingredient, or close-up proof |
| `person.jpg` | Expert, founder, or customer quote |
| `offer.png` | Final offer/product bundle image |

All images must be local relative paths in built pages.
