# Homepage Section Library

Every section here is extracted from `hp-01-brand-story/index.html` — battle-tested on KURA.

Each folder contains one or more HTML snippet files. Open them standalone to preview, copy-paste into any template when building.

---

## How to Use

1. Open the section file you need
2. Copy the HTML between the section separator comments
3. Paste into your target template
4. Copy the relevant CSS from `:root` and the section's CSS block
5. Replace `[BRAND]` tokens with real content

## Background Rhythm Rule

When assembling sections, never stack two sections with the same background. Recommended order:

```
cream → slate → white → cream → slate → white → offblack → slate
```

This gives each section visual separation without borders.
